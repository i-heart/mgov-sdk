# PHP

```
┌── FileUploadApplication.php   # 실행 예제 파일
├── FileUploadClient.php        # handleFileUpload 함수를 제공하는 파일
└── SimpleHttpClient.php        # HTTP 관련 클래스 (HTTP request에 대한 외부 의존 제거를 위함)
```

---

## 샘플 실행
```
(FileUploadApplication.php 파일을 환경에 맞게 편집 및 저장한 후)
$ php FileUploadApplication.php
```
